# Lavish Verma Portfolio 🍷

Link -> https://lavish-portfolio-six.vercel.app/

A clean, beautiful, and responsive portfolio template for showcasing your wine brand.

## Features

- Responsive design
- Easy content and image customization
- Modern React stack

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (v6 or higher)
- [Git](https://git-scm.com/)

### Installation

```bash
git clone <your-repo-url>
cd developerFolio
npm install
```

### Customization

- Edit `public/profile.json` and `src/portfolio.js` for your wine brand details.
- Replace images in `public/` and `src/assets/images/` with your own.
- Update colors in `src/_globalColor.scss` if desired.

## Building

```bash
npm run build
```
This creates a production-ready `build/` folder.

## Deployment

### Deploying to Vercel (Recommended)

1. Go to [vercel.com](https://vercel.com/) and sign up.
2. Import your GitHub repository or drag-and-drop your folder.
3. Vercel auto-detects React and builds your project.
4. Your site will be live at a Vercel URL.


Copy `env.example` to `.env` and update any required keys.

```bash
# For Windows
copy env.example .env
```
